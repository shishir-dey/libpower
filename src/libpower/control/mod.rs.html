1//! # Digital Control Systems
2//!
3//! This module provides a comprehensive collection of digital control algorithms commonly
4//! used in power electronics and embedded control systems. All controllers are implemented
5//! as discrete-time systems suitable for real-time control loops.
6//!
7//! ## Available Controllers
8//!
9//! ### Linear Controllers
10//! - [`cntl_pi`] - Proportional-Integral (PI) controller with anti-windup
11//! - [`cntl_pid`] - Proportional-Integral-Derivative (PID) controller
12//!
13//! ### Advanced Controllers
14//! - [`cntl_2p2z`] - Two-pole, two-zero controller (2P2Z compensator)
15//! - [`cntl_3p3z`] - Three-pole, three-zero controller (3P3Z compensator)
16//!
17//! ## Controller Selection Guide
18//!
19//! ### PI Controller
20//! - **Best for**: DC-DC converters, current loops, simple voltage regulation
21//! - **Advantages**: Zero steady-state error, built-in anti-windup
22//! - **Limitations**: Cannot add zeros for improved transient response
23//!
24//! ### PID Controller
25//! - **Best for**: General-purpose control, systems requiring derivative action
26//! - **Advantages**: Includes derivative term for improved transient response
27//! - **Limitations**: Sensitive to noise, requires tuning of three parameters
28//!
29//! ### 2P2Z Controller
30//! - **Best for**: DC-DC converter voltage loops, single-phase power supplies
31//! - **Advantages**: Two zeros for improved transient response and stability margins
32//! - **Applications**: Buck/boost converters, battery chargers
33//!
34//! ### 3P3Z Controller
35//! - **Best for**: Three-phase systems, complex power conversion topologies
36//! - **Advantages**: Additional poles and zeros for precise frequency response shaping
37//! - **Applications**: Grid-tied inverters, motor drives, advanced power supplies
38//!
39//! ## Design Considerations
40//!
41//! ### Sampling Frequency
42//! All controllers assume a fixed sampling period. Typical sampling frequencies:
43//! - Current loops: 10-100 kHz
44//! - Voltage loops: 1-10 kHz
45//! - Outer control loops: 100 Hz - 1 kHz
46//!
47//! ### Saturation and Anti-windup
48//! All controllers include configurable output saturation limits to prevent:
49//! - Integrator windup
50//! - Actuator saturation
51//! - System instability
52//!
53//! ### Coefficient Calculation
54//! Controller coefficients are typically designed using:
55//! - Bode plot analysis
56//! - Root locus techniques
57//! - Digital control design tools (MATLAB, Python control)
58//!
59//! ## Example Usage
60//!
61//! ```rust
62//! use libpower::control::cntl_pi::ControllerPI;
63//!
64//! // Create PI controller for current loop
65//! let mut current_controller = ControllerPI::with_gains(0.5, 100.0);
66//! current_controller.set_limits(-1.0, 1.0); // Â±1.0 duty cycle
67//!
68//! // Control loop execution
69//! let current_reference = 5.0; // 5A reference
70//! let current_measurement = 4.8; // 4.8A measured
71//! let duty_cycle = current_controller.calculate(current_reference, current_measurement);
72//! ```
