Struct EventMapper 

Source 
pub struct EventMapper<'a, E, F, LocalId> { /* private fields */ }
Expand description

Zero-cost adapter that remaps event IDs from one layer into another.

This lets a local layer emit local event IDs into a shared outer event bus without coupling the local layer to the outer ID type.
