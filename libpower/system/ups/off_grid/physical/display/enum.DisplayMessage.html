Expand description

Display control trait and implementations for UPS physical components.

This module provides a trait for display control with two text buffers (lines), along with a default mock implementation. The display is based on the GLCD system from the assembly code, supporting various messages and conditions. Display message types based on the assembly LCD_DISPLAY functions. For 20x4 LCD display.
